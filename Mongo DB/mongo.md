
# Language | Mongo Shell


## Abrir **(en mi caso...)**

1. Abrir un terminal y poner `sudo mongod`
2. Abrir otro terminal y poner `cd..` y luego `cd ..`
3. Dentro `cd var/log` 
4. Y ejecutar `mongo`

## Comandos

- Muestra todas las bds
```
    show dbs
``` 
- Usar una db
```
    use movies
```
